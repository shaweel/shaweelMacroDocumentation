# shaweelMacro
shaweelMacro is a [Bee Swarm Simulator](https://www.roblox.com/games/1537690962) macro for [GNU](https://gnu.org/home.en.html)+[Linux](https://kernel.org) and [Windows](https://windows.com).

The main focuses of this macro are reducing time not gathering(looting faster, better pathing) and running in an isolated environment.
